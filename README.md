# How to Run

Installation requirements.

```
virtualenv env

```

```
source env/bin/activate
```

pip3 install -r requirements.txt

```

## Sample Chatbot

The required files must be downloaded first, so it takes a while.

```

cd sample

```

```

python3 sample.py

```

## Chatbot Adapters

The list of adapters is inside a adapters_list.py

```

cd adapters

```

```

python3 adapters.py

```

```
