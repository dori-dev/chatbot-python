# Chatbot Python

Chatbot with python!

#

# How to Run

Installation requirements.

```
pip3 install -r requirements.txt
```

## Sample Chatbot

The required files must be downloaded first, so it takes a while.

```
cd sample
```

```
python3 sample.py
```

## Chatbot Adapters

The list of adapters is inside a adapters_list.py

```
python3 adapters.py
```
